Add Capsule CRM to Cursor

Follow these steps to add the Capsule CRM MCP server to your Cursor settings:

Setup Instructions

  1. First, install the MCP server: 
    git clone https://github.com/fuzzylabs/capsule-mcp.git
cd capsule-mcp
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -e .
  2. Get your Capsule API token:
    • Log into your Capsule CRM account
    • Go to My Preferences â†’ API Authentication
    • Create a new API token and copy it
  3. Configure Cursor:
    • Open Cursor's settings (Cmd + , on macOS, Ctrl + , on Windows/Linux)
    • Search for "MCP Servers" in the settings
    • Add the following configuration:
    {
  "capsule-crm": {
    "command": "/usr/bin/python3",
    "args": [
      "/path/to/your/capsule-mcp/capsule_mcp/server.py"
    ],
    "env": {
      "CAPSULE_API_TOKEN": "your_capsule_api_token_here"
    }
  }
}
  4. Update the paths:
    • Replace /usr/bin/python3 with your Python path (find it with which python3)
    • Replace /path/to/your/capsule-mcp/ with the actual path to your installation
    • Replace your_capsule_api_token_here with your actual API token
  5. Restart Cursor to activate the MCP server
