Add Capsule CRM to Cursor

Click the button below to add the Capsule CRM MCP server to your Cursor settings:

Manual Setup

If the button doesn't work, you can add the server manually:

  1. Open Cursor's settings (Cmd + , on macOS, Ctrl + , on Windows/Linux)
  2. Search for "MCP" in the settings search bar
  3. Under "MCP Servers", click "Add Server" and enter:
    • Name: Capsule CRM
    • URL: http://localhost:8000/mcp
  4. Click "Save" and restart Cursor
